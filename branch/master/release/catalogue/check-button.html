check-button
Чекбокс в виде кнопки

Для того, чтобы Яндекс выглядел аккуратнее и развивался в одном стиле, были разработаны декоративные (нестандартные для операционной системы) элементы форм.

Контролы реализованы на CSS3, в MSIE8 и ниже контролы деградируют до нативных. Только bemhtml реализация.

Модификаторы

checked

Состояние чекбокса — активное, чекбокс вырбан (кнопка вжата)
yesВключено

disabled

Неактивное состояние. Нажать на чекбокс-кнопку нельзя.
yesВключено

focused

Cостояние в фокусе
yesВключено

only-icon

Чекбокс-кнопкой только с иконкой
yesВключено

pressed

Визуальное нажатие кнопки — чекбокс не вырбан
yesВключено

size

Размер
mРазмер шрифта 15px
sРазмер шрифта 13px

theme

Темы оформления

Элементы

control

Нативный чекбокс

icon

Используется для отображения иконки в чекбоксе, рядом с текстом или самостоятельно (требует указания модификатора _only-icon)

text

Обрамляет текст чекбокса-кнопкой, используется в нажатом состоянии для сдвига текста без смещения рядом стоящего текста
