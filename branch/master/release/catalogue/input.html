input
Поле ввода с шириной 100%

Для того, чтобы Яндекс выглядел аккуратнее и развивался в одном стиле, были разработаны декоративные (нестандартные для операционной системы) элементы форм.

Контролы реализованы на CSS3, в MSIE8 и ниже контролы деградируют до нативных. Только bemhtml реализация.

Разновидности полей ввода:

  • простое;
  • с меткой;
  • с подсказкой;
  • с сообщением об ошибке;
  • с примерами;
  • с кнопкой очистки;
  • поле ввода текстовой областью — форма для набора текста;
  • c автокомплитом.
