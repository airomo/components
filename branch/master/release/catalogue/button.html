button
Кнопка

Для того, чтобы Яндекс выглядел аккуратнее и развивался в одном стиле, были разработаны декоративные (нестандартные для операционной системы) элементы форм. Одним из таких элементов является кнопка.

Контролы реализованы на CSS3, в MSIE8 и ниже контролы деградируют до нативных. Только bemhtml реализация.

Разновидности кнопок

  • Обычная — все кнопки, используемые в формах;
  • Кнопка действий — кнопка, отправляющая форму на сервер;
  • Ссылкой — может быть только обычной;
  • С тенью — летающие кнопки, с большой тенью, используются, например, над картой;

Кнопки с иконками

Вот эти кнопки

Иконки, можно использовать только в кнопках-ссылках. Это связано с тем, что в обычных кнопках content из входных данных копируется в value нативного контрола.

API контрола

  • Общее API контролов.

Блок button — является кнопкой. Используется в других блоках.

В простом случае для отображения блока необходимо задать следующий BEMJSON:

{
   block: 'button',
   content: 'Текст кнопки'
}
